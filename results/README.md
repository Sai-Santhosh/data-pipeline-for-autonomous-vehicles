# Results

This folder stores pipeline outputs and analysis results.

- **Notebook outputs**: Run the notebooks in `../notebooks/` and export figures or tables here if needed.
- **Dashboard screenshots**: Optional screenshots of the Streamlit dashboard for documentation.
- **CSV exports**: Run `python scripts/export_results.py` from repo root (with DB populated) to generate:
  - `miles_per_intervention_sample.csv`
  - `alerts_sample.csv`
  - `interventions_per_vehicle_sample.csv`
